## Types

* ### Number
  A floating point number.
  >*name* = *123*
* ### String
  >*name*__$__ = *"something"*  
  >*name*__$__ = *"Line1*  
  >*line2"*  
  >*name*__$__ = *"escape quotes ""like this"""*  
* ### Array
  A list of values of any type.
  >*name*__#__ = *{1,2,3}*
  
  Falsey values are `0`, `""`, and `{}`
